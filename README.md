# Team Octagon Website

A simple and responsive team portfolio website created using HTML, CSS, and JavaScript.

## Live Website

https://octagon-beta.vercel.app/

## Features

- Responsive design
- Team introduction
- Team members section
- Smooth navigation
- Modern user interface

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/angila-04/octagon.git
   ```
2. Open the project folder.
3. Open `index.html` in your browser.
